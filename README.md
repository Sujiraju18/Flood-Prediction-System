# 🌊 Flood Prediction System

A Machine Learning based web application that predicts flood risk using environmental and weather-related parameters.

---

## 📌 Project Overview

Floods are among the most destructive natural disasters. This project uses Machine Learning algorithms to predict whether an area has a high or low flood risk based on several environmental factors.

The application is built using Python, Flask, Scikit-learn, and XGBoost.

---

## 🚀 Features

- Predict flood risk instantly
- User-friendly web interface
- Machine Learning based prediction
- Responsive UI
- Model trained on flood prediction dataset

## ▶ How to Run

Clone the repository

```
git clone <repository-url>
```

Create virtual environment

```
python -m venv venv
```

Activate virtual environment

Windows

```
venv\Scripts\activate
```

Install dependencies

```
pip install -r requirements.txt
```

Run Flask

```
python app.py
```

Open

```
http://127.0.0.1:5000
```
